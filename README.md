# Deep Das — Financial Modelling Portfolio

## The Coca-Cola Company (KO) — 3-Statement Financial Model

### Files in this repository
- KO_Model_Clean.xlsx — Integrated 3-statement financial model built in Microsoft Excel
- KO_Dashboard_Final.pdf — Power BI financial dashboard

### About the model
- Integrated Income Statement, Balance Sheet and Cash Flow Statement
- Actuals 2022A to 2025A sourced from KO 10-K filings
- Forecasts 2026F and 2027F driven by scenario analysis
- Three scenarios: Bear, Base and Bull
- Dynamic share buyback mechanism balancing the Balance Sheet without a plug
- Connected to Power BI dashboard via OneDrive

### Tools used
Microsoft Excel, Microsoft Power BI, KO 10-K Annual Reports
